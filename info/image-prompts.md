# Image Prompts for VetTop

Generated for AI image tools (Midjourney, DALL-E, Flux, etc.)

## Tips
- Generate all images in one category before moving to the next for visual consistency
- Use the same seed/style reference across a category if your tool supports it
- JPEG or WebP at 80% quality
- Save images to the project's `public/` directory matching the exact paths listed
- For team portraits, use consistent lighting and framing across all 4 photos
- Gallery images 1-3 are also used in the hero section circles — pick strong subjects that work cropped circular
